  購入したいものをランキング形式で提示して無駄遣いを減らすサービス

  <br>

  ## HoshiMeを作成した背景
  ずっと欲しかったものがあったにも関わらず、買い物に行った際に違うものを衝動買いしてお金を使ってしまい、
  本当に欲しかったものを買うためのお金がなくなってしまったことがありました。
  優先順位を自動で算出し提示してくれるようなサービスがあったらいいなと思い、HoshiMeをリリースしました！

  <br>

  ## HoshiMeを勧めたいユーザー
  💰 爆買いしてしまう人
  💰 欲しいものの優先順位がわからない人
  💰 無駄な出費を抑えたい人
  💰 お金と時間を有効活用したい人

  <br>

  ## HoshiMeで解決できること
  衝動買いや無駄遣いが減って、本当に欲しいものを優先的に購入することができるようになります。
